\subsubsection{READ}
\label{sec:rest:api:schools:classes:read}
Es sind nur Anfragen mit der HTTP-GET-Methode für ein READ auf die Daten zugelassen.
Der Endpunkt gibt eine Liste von Referenz-IDs für Klassen an der Schule zurück welche unter /api/classes/\$id abgerufen werden können.

Die Antwort erfolgt mit dem JSON-Objekt \reflisting{lst:code:rest:api:schools:classes:read:ret}. 
Die einzelnen Felder der Antwort werden in \reftabllec{tab:rest:api:schools:classes:read:ret:json} beschrieben.
Die Berechtigungen auf den Endpunkt können \reftabllec{tab:rest:api:schools:classes:read:right} entnommen werden.

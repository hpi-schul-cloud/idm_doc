\subsubsection{READ}
\label{sec:rest:api:schools:subjects:read}
Es sind nur Anfragen mit der HTTP-GET-Methode für ein READ auf die Daten zugelassen.
Der Endpunkt gibt eine liste von Referenz-IDs für Unterrichtsfächern zurück welche unter /api/subjects/\$id abgerufen werden können.

Die Antwort erfolgt mit dem JSON-Objekt \reflisting{lst:code:rest:api:schools:subjects:read:ret}. 
Die einzelnen Felder der Antwort werden in \reftabllec{tab:rest:api:schools:subjects:read:ret:json} beschrieben.
Die Berechtigungen auf den Endpunkt können \reftabllec{tab:rest:api:schools:subjects:read:right} entnommen werden.

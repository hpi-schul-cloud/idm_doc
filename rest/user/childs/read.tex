\subsubsection{READ}
\label{sec:rest:api:user:childs:read}
Es sind nur Anfragen mit der HTTP-GET-Methode für ein READ auf die Daten zugelassen.
Die Route gibt für den eingeloggten Benutzer die Liste mit den IDs seiner Kinder wieder.
Die Antwort erfolgt mit dem JSON-Objekt \reflisting{lst:code:rest:api:user:childs:read:ret}. 
Die einzelnen Felder der Antwort werden in Tabelle \reftabllec{tab:rest:api:user:childs:read:ret} beschrieben.
Die Berechtigungen auf den Endpoint können \reftabllec{tab:rest:api:user:childs:read:right} entnommen werden.
